Replaces the last space in the title with a non-breaking space.

== Description ==

Lightweight plugin that replaces the last space in the main query titles with a non-breaking space.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/unwidow-title` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin though the `Plugins` screen in WordPress
